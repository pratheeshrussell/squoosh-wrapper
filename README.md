# Squoosh Library Wrapper

> [!NOTE]  
> Everything in this repo was generated by Gemini 3.1 and Claude opus in antigravity

A browser-focused wrapper around the WASM image codecs from Google's [Squoosh](https://github.com/GoogleChromeLabs/squoosh) project. This library allows you to compress, resize, and convert images entirely in the browser using the same high-quality codecs that power the Squoosh web app.

## Features

- **No build required**: The WASM binaries are pre-built and ready to use.
- **Lazy loading**: Each codec's WASM binary is only fetched when you first use it.
- **Browser-first**: Designed for the web. Uses native `Canvas` / `createImageBitmap` for decoding where possible, and WASM for encoding.
- **ES Modules**: Clean, tree-shakeable API.
- **Feature Detection**: Automatically uses SIMD and Multi-Threaded WASM variants if the browser supports them.

## Supported Codecs

| Codec | Encode | Decode | Notes |
|-------|--------|--------|-------|
| **MozJPEG** | ✅ | ❌ | High-quality JPEG encoder |
| **WebP** | ✅ | ✅ | Supports SIMD auto-detection |
| **AVIF** | ✅ | ✅ | Supports Multi-Threading auto-detection |
| **PNG** | ✅ | ✅ | Lossless PNG optimization |
| **Resize** | ✅ (Process)| — | High-quality resizing (Lanczos, Mitchell, Catrom, Triangle) |

> Note: To decode JPEGs, standard PNGs, or other formats your browser natively supports, use the `nativeDecode` helper included in this library.

## Installation / Setup

1. Copy the `src` and `codecs` folders into your project.
2. Ensure your web server serves `.wasm` files with the `application/wasm` MIME type.
3. Import the functions you need from `src/index.js`.

> **Building from source:** A GitHub Action is included (`.github/workflows/build-codecs.yml`) to let you build the WASM binaries from the original C++/Rust source code using Docker.

## Usage

### Decoding

Use the native browser decoder to convert an image file (e.g., from an `<input type="file">`) into raw `ImageData` that the encoders can process:

```javascript
import { nativeDecode } from './src/index.js';

// Decode a File or Blob
const file = document.getElementById('fileInput').files[0];
const imageData = await nativeDecode.decode(file);

// Or decode from a URL
const imageData2 = await nativeDecode.decodeUrl('https://example.com/image.jpg');
```

### Encoding

```javascript
import { mozjpeg, webp, avif, png } from './src/index.js';

// MozJPEG (default quality: 75)
const jpegBuffer = await mozjpeg.encode(imageData, { quality: 80 });

// WebP (auto-detects SIMD)
const webpBuffer = await webp.encode(imageData, { quality: 75 });

// AVIF (auto-detects threads)
const avifBuffer = await avif.encode(imageData, { quality: 50 });

// PNG (Lossless)
const pngBuffer = await png.encode(imageData);
```

### Resizing

```javascript
import { resize } from './src/index.js';

const resizedImageData = await resize.resize(imageData, {
  width: 800,
  height: 600,
  method: 'lanczos3' // or 'mitchell', 'catrom', 'triangle'
});
```

### Displaying the Result

Convert the output `ArrayBuffer` back to a Blob to display it or trigger a download:

```javascript
const blob = new Blob([jpegBuffer], { type: 'image/jpeg' });
const url = URL.createObjectURL(blob);

// Show in an image tag
document.getElementById('output').src = url;

// Or trigger download
const a = document.createElement('a');
a.href = url;
a.download = 'compressed.jpg';
a.click();
```
